** FLUXO DE TRABALHO

1. criar o diretorio do projeto
2. iniciar o git entro do diretorio criado (git init .)
3. edita ou cria arquivos
4. git status
5. git add <nome do arquivo>
6. git status
7. git commit -m "msg"
8. git status
9. volta ao passo 3

git diff . (detalha oq foi modificado)

git reset (volta um estado de commit)
git checkout -b nome da branch (cria ramificação)
git branch (mostra as branchs)
git checkout master/main (volta para principal) -b (novo)
git merge (junta as branches)
git branch -d nome do branch (apagar uma branch)
